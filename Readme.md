# チートシート

 .
 スペースつけて .でディレクトリ一括指定

- *git init*
gitとしての使用を開始

- *git add {file}*
コミット準備

- *git commit -m {msg}*
コミット時のコメント

- *git --version*
gitのversionを表示

- *git log*
gitのログを見る

- *git config-global user.name {name}*
gitでの署名を変更

- *git status(-s)*
gitでの更新を確認
